# BRUTALIST.OS

Raw. Unfiltered. Brutal.

A brutalist design portfolio website featuring high-contrast aesthetics, exposed HTML structure, and zero framework dependencies.

## Features

- **Pure HTML/CSS/JavaScript** - No frameworks, no libraries
- **7 Interactive Sections** - Home, About, Work, Services, Lab, Contact, Manifesto
- **Custom Cursor** - Smooth cursor tracking with hover effects
- **Glitch Animations** - CSS-based glitch text effects
- **Terminal Emulator** - Fake terminal interface for brutalist aesthetic
- **Interactive Lab** - Real-time CSS playground with sliders and controls
- **Canvas Background** - Animated particle system
- **Responsive Design** - Mobile-friendly with hamburger menu
- **Keyboard Navigation** - 1-7 keys to navigate sections, Esc for home

## Design Philosophy

- **Honesty in Structure** - Exposing raw HTML divs and borders
- **Function Over Decoration** - Every pixel serves a purpose
- **Maximum Contrast** - Black, white, neon red, electric blue, acid yellow

## Stack

- HTML5
- CSS3 (Custom Properties, Grid, Animations)
- Vanilla JavaScript
- Google Fonts (Space Mono, Syne)
- Canvas API (background particles)

## Sections

1. **Home** - Hero with glitch text, latest experiments, marquee
2. **About** - Company history, brutalist philosophy timeline
3. **Work** - Project gallery with hover effects
4. **Services** - 3-tier pricing (Raw $5K, Brutal $12K, Concrete $25K)
5. **Lab** - Interactive CSS playground with live preview
6. **Contact** - Contact form and direct contact info
7. **Manifesto** - 3 brutalist design principles
8. **Secret** - Hidden 404 page accessible via footer

## Interactions

- **Color Switcher** - Fixed position accent color picker
- **Notification System** - Toast notifications for all actions
- **Loading Screen** - Fake system initialization animation
- **Card Rotations** - Click cards to rotate
- **Counter Lab** - Click counter with achievement at 10
- **Hover Effects** - Color shifts, scale transforms, overlay reveals
